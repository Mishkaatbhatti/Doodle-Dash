# Doodle Dash ✨🎮

Doodle Dash is an interactive gesture-controlled web application that combines air drawing, shape connection challenges, and immersive visual effects using real-time hand tracking technology. The project allows users to interact with digital visuals entirely through hand movements captured via webcam.

The application features multiple game modes designed to enhance creativity, coordination, and gesture-based interaction in a fun and engaging environment.

---

# 🚀 Features

* ✋ Real-time hand gesture tracking
* 🎨 Air drawing and doodling
* 🧩 Shape connection challenge mode
* 🌌 Interactive double-hand constellation effects
* 🎵 Audio synthesis and interactive sound effects
* 🌈 Dynamic visual animations
* 💻 Browser-based experience
* ⚡ Lightweight and responsive design

---

# 🎮 Game Modes

## 1️⃣ Air Doodle Mode

A free-hand drawing mode where users can draw anything in the air using hand gestures.

### Features

* Free air drawing
* Real-time gesture interaction
* Dynamic visual feedback
* Smooth drawing experience

Users can unleash their creativity by sketching shapes, patterns, or artwork directly in the virtual space.

---

## 2️⃣ Hand Connect Mode

A gesture-based puzzle mode where users connect numbered points in sequence to create predefined shapes.

### Gameplay

* Connect points in order: **1 → 2 → 3 → 4**
* Each level presents a different target shape
* Shapes increase in complexity with progression

### Levels

* ⭐ Level 1: Star Shape
* ❤️ Level 2: Heart Shape
* 🔷 Additional levels with unique patterns

This mode improves hand coordination and interactive engagement through gesture-controlled gameplay.

---

## 3️⃣ Constellation – Double Hand VFX Mode

An immersive visual effects mode that uses both hands simultaneously to create interactive constellation-style visual patterns.

### Features

* Double-hand gesture tracking
* Dual-hand air drawing
* Dynamic particle and visual effects
* Interactive constellation-inspired visuals

This mode focuses more on visual interaction and artistic effects rather than structured gameplay.

# ⚙️ Installation & Usage

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Mishkaatbhatti/Doodle-Dash.git
```

## 2️⃣ Navigate to the Project Folder

```bash
cd Doodle-Dash
```

## 3️⃣ Run the Application Using a Local Server

Start a local Python server:

```bash
python -m http.server 8000
```

Then open your browser and visit:

```bash
http://localhost:8000
```

> Make sure webcam permissions are enabled in your browser for gesture tracking to function properly.
